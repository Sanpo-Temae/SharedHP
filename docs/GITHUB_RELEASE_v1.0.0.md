# SharedHP 1.0.0

SharedHP の初回正式リリースです。

## 重要

- **Paper専用**プラグインです。
- **Spigot / Bukkit / CraftBukkit は非対応**です。
- 対応環境は **Paper 1.21.11 / Java 21** です。
- サーバーの `plugins` フォルダには、SharedHP のJarを **1つだけ** 入れてください。
- 更新時は `/reload` ではなく、`stop -> Jar入れ替え -> 起動` を推奨します。

## ダウンロード

`SharedHP-1.0.0.jar` をダウンロードし、Paperサーバーの `plugins` フォルダに入れてください。

## 主な機能

- 任意人数の参加者が1つの共有HPを持つ
- ダメージ共有
- 回復効率25%、`/sharedhp heal <percent>` で変更可能
- ボスバー表示
- 吸収ハート無効化
- 被ダメージランキング
- config保存
